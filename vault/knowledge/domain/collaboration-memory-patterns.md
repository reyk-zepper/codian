# Collaboration Memory Patterns

## Muster

Wenn der User nach einem funktionierenden Ausbau sehr kurze Freigaben wie "mach das" oder "mach alles das was du brauchst" gibt, ist autonome Weiterarbeit mit konkreten Artefakten wertvoller als weitere Meta-Diskussion.

## Ableitung fuer Codex

- Fortschritt lieber durch echte Notes, echte Retrieval-Pfade und echte Workflow-Helfer erzeugen
- Dauerhafte Erkenntnisse sofort in Codian schreiben
- Nach einem Infrastruktur-Ausbau schnell in Nutzungsmodus wechseln
- Den Nutzen fuer kuenftige Sessions ueber gut verlinkte, leicht auffindbare Notes maximieren
