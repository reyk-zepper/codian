# Obsidian Vault Pattern

An Obsidian-based memory vault provides a practical human-readable interface for long-term AI memory.

The core pattern is:

1. store durable knowledge as markdown notes
2. keep a curated index as the retrieval entrypoint
3. use wikilinks for graph navigation
4. validate structural health with repeatable scripts
